# jtfg-eup-core

Starter scaffold for a Node.js module.

## Usage

1. Copy your module files into `src/`.
2. Update `package.json` metadata (`name`, `description`, `author`, version, etc.).
3. Run `npm install` if you add dependencies.

## Development

- Entry point: `src/index.js`
- Tests: add your preferred test framework and update `scripts.test`
